Installation notes

To install, pay attention to the dependencies :
0/ Install Module Manager if not already done yet, via CKAN or https://forum.kerbalspaceprogram.com/index.php?/topic/50533-141-module-manager-307-may-5th-2018-its-dangerous-to-go-alone-take-those-cats-with-you/
1/ Install Textures Unlimited via CKAN or here https://github.com/shadowmage45/TexturesUnlimited/releases
2/ Install TURD. There is no CKAN nor Spacedok nor Github availability, you have to download it from here: https://forum.kerbalspaceprogram.com/index.php?/topic/174188-111x-textures-unlimited-recolour-depot/
3/ Install this patch with simply matching Gamedata folders from the archive and your KSP installation
